Jekyll is a static site generator that works especially well for blogs.

To get started locally:

1. Install dependencies with `bundle install`
2. Start the site with `bundle exec jekyll serve`
3. Open `http://localhost:4000`

Once running, you can add posts in the `_posts/` directory using the `YYYY-MM-DD-title.md` naming format.
